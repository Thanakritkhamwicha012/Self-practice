ผมนำโจทย์ที่ยังทำไม่สำเร็จในชั้นเรียนกลับมาลองทำใหม่อีกครั้ง ซึ่งช่วยให้เข้าใจทั้งวิธีเขียนโค้ดและวิธีคิดมากขึ้น โดยโค้ดนี้เป็น **function สำหรับนับจำนวนคำซ้ำในประโยค** และคืนค่าออกมาเป็น object
* **การรับค่าเข้า** → ฟังก์ชันจะรับเฉพาะค่าประเภท `string` เท่านั้น ถ้าไม่ใช่ string จะคืนค่า `undefined`
* **การจัดการข้อความ** → ใช้ `.toLowerCase()` เพื่อแปลงข้อความทั้งหมดให้เป็นตัวพิมพ์เล็ก และใช้ `.split(" ")` เพื่อแยก string ออกเป็น array ตามช่องว่าง
* **การสร้างตัวเก็บข้อมูล** → กำหนดตัวแปร object ว่างไว้เพื่อเก็บคำและจำนวนครั้งที่ปรากฏ
* **การวนลูปเช็คคำ** → ใช้ `for...of` ในการวนลูปแต่ละคำใน array
  * ถ้าคำนั้นเคยมีใน object แล้ว → ให้นับเพิ่มขึ้นทีละ 1
  * ถ้าเป็นคำใหม่ → กำหนดค่าเริ่มต้นเป็น 1
สุดท้าย object ที่ได้จะมี key เป็น “คำ” และ value เป็น “จำนวนครั้งที่คำนั้นเจอในประโยค”
