ผมใช้ ChatGPT เพื่อช่วยอธิบายว่าโค้ดที่เขียนทำงานอย่างไร และทำให้ผมเข้าใจโครงสร้างของโค้ดมากขึ้น โดยโค้ดที่เขียนเป็น ฟังก์ชันสำหรับตรวจสอบว่า object สองตัวมีค่าเหมือนกันหรือไม่ ซึ่งจะตรวจจากทั้ง key และ value
1. การตรวจสอบรอบแรก – ฟังก์ชันจะเช็คก่อนว่าทุกอย่างเหมือนกันหรือไม่ ถ้าใช่จะคืนค่า `true` และหยุดทำงานทันที
2. การตรวจสอบรอบสอง – จะเช็คเฉพาะว่าเป็น type `object` เท่านั้น และไม่ใช่ค่าว่าง
3. สร้างตัวแปรเก็บ key – เพื่อดึง key ของทั้งสอง object มาเปรียบเทียบกัน
4. เช็คความยาวของ key – ถ้า key มีจำนวนไม่เท่ากัน แปลว่า object ไม่เหมือนกัน
5. ตรวจสอบ value – ใช้ `for...of` loop เพื่อวนดูค่าของ object แล้วเปรียบเทียบกัน
